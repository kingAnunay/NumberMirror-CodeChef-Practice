# NumberMirror-CodeChef-Practice
CodeChef NumberMirror Practice


NUMBER MIRROR

PROGRAM:
Write a program that accepts a number, n, and outputs the same.

INPUT:
The only line contains a single integer.

OUTPUT:
Output the answer in a single line

CONSTRAINTS:
0 ≤ n ≤ 10^5

SAMPLE INPUT:
123

SAMPLE OUTPUT:
123
